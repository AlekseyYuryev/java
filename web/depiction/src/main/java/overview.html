JpegRDF manipulates RDF metadata associated with JPEG images.

Overview

JpegRDF is a command-line utility to add, extract, update, and query RDF metadata stored in the comment section of JPEG images. See the reference page for more details.

Quick Start

You must have the jpegrdf.jar and the Jena 1.6.x API jar files on your CLASSPATH.

Use -h for help:

java com.nwalsh.jpegrdf.JpegRDF -h

Start with a JPEG image, myimage.jpg. If it includes EXIF data (e.g., if you created it with a digital camera), convert that data into RDF:

java com.nwalsh.jpegrdf.JpegRDF -exif myimage.jpg

View the results:

java com.nwalsh.jpegrdf.JpegRDF -s myimage.jpg

Add your own metadata:

java com.nwalsh.jpegrdf.JpegRDF -ns dc http://purl.org/dc/elements/1.1/ -al dc:description "A test image" myimage.jpg

If you use a namespace often, you can simply the command by adding that namespace to your configuration file (~/.jpegrdf.rdf). Then you can simply use the prefix you declared:

java com.nwalsh.jpegrdf.JpegRDF -al dc:creator "John Doe" myimage.jpg

Find all the images created by John Doe:

java com.nwalsh.jpegrdf.JpegRDF -ql dc:creator "John Doe" *.jpg
